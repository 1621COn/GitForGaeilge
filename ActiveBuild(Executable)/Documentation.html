GitForGaelige is a opensource project that started with the love and passion for other languages. With the belief that knowledge should be free and easily accessible for everyone. This program is for the spoken Irish language (see more in the 'index.html' tab.)
This documentation will not feature how-to use C sharp with Visual Studio. The program is featured currently with basic language such Hi's, Hello's, numbers, months, Combined with these words, audio was included for pronunciation; additionally phonetics are added to the words. the green button is how you would play the audio, the index.html is another html file that tells the actual history of the usage of the language and more specifically why this project exists.
